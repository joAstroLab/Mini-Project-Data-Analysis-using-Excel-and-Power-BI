# Mini Project - Data Analysis using Excel & Power BI
This document presents a mini project on data analysis conducted using Excel and Power BI, focusing on a video game sales dataset containing over 64,000 records with no duplicates. The project aims to analyze sales trends across different game genres—such as action, sports, puzzle and adventure—to understand their impact on game popularity and identify top-performing categories. It details a comprehensive data cleaning process, including removing irrelevant columns, standardizing headers, handling missing values, formatting data and creating new fields like the year column. The analysis uses pivot tables to evaluate total global sales by genre, revealing that genres like Misc, Action and Shooter are the most popular, while Shooter, Sandbox and Sports emerge as the top-performing genres based on average sales. Overall, the project demonstrates effective data preparation and visualization techniques to extract meaningful business insights from raw data.
